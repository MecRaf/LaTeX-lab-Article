# Artículo de laboratorio (Spanish laboratory article)

Short description
A short Spanish-language laboratory article and its LaTeX source. This repository contains the .cls class file, .tex source, data files and images.

Repository layout
- paper/ — document.tex and MiArticulo.cls
- data/ — .dat files used in paper
- images/ — PNG figures
- output/ — compiled PDF

How to compile (locally)
1. Install TeX Live (or MikTeX).
2. From repository root:
   cd paper
   pdflatex document.tex
   pdflatex document.tex
3. The PDF will be `document.pdf`.
